Запуск(контейнер приложения и БД):
```
docker-compose up --build
```
Затем открываем запущенный контейнер приложения интерактивно в другом терминале:
```
docker attach javaastonsteptwo-app-1
```
После запуска вводим число 4 (вывод таблицы БД), чтобы меню приложения отрисовалось вновь.
Хранение логов в logs
